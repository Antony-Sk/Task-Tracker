# Task Tracker
Web api for manage your projects and tasks!
## Used technologies
 - Java
 - Maven
 - Spring boot
 - Hibernate
 - PostgreSQL
 - JPA
 - Lombok
 - Swagger Open API
## Documentation
http://localhost:20000/swagger-ui/index.html#/task-controller/deleteTask
## How to build the project
```
$ git clone https://github.com/Antony-Sk/Task-Tracker
$ cd TaskTracker
$ docker-compose up
```
