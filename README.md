# Task Tracker
Web api for manage your projects and tasks!
## Used technologies
 - Java
 - Maven
 - Spring boot
 - Hibernate
 - PostgreSQL
 - JPA
 - Lombok
 - Swagger Open API
## Documentation
https://task-tracker-test-task.herokuapp.com/swagger-ui/index.html#/
## How to build the project
```
$ git clone https://github.com/Antony-Sk/Task-Tracker
$ cd TaskTracker
$ docker-compose up
```
